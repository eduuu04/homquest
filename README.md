# 🏡 HomQuest - Sistema de Gamificación del Hogar

HomQuest es una aplicación móvil y web diseñada para convertir las tareas del hogar en un juego divertido con puntos (XP), niveles, rachas, logros y tienda de recompensas familiares.

## ✨ Características Principales
- 🎮 **Gamificación Completa**: Ganancia de XP, niveles personalizados, rachas y logros por tareas cumplidas.
- 🛍️ **Tienda de Recompensas**: Canje de monedas (HomCoins) por recompensas familiares definidas por los administradores.
- ⚡ **Inicio de Sesión Automático**: Mantén la sesión iniciada en tu dispositivo de forma permanente.
- 👑 **Panel de Administración**: Gestión de tareas, aprobación con fotografías, niveles y miembros del hogar.
- 📲 **Capacitor Mobile Native Support**: Soporte completo para Android.
